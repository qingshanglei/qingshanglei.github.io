## 批量修改文件名：

```
dir /b 文件名 >names.txt //批量获取文件名
ren 旧名 新名 // 批量修改文件名
```

​    使用slsx表格替换要修改的名字 （Ctrl+H：替换）

​    双击slsx表格单元格右下角，复制当前行的所有行。
